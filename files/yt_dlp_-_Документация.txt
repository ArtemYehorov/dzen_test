-f FORMAT — выбрать формат видео/аудио. Пример: -f bestaudio

-o ШАБЛОН — задать шаблон имени файла. Пример: -o "%(title)s.%(ext)s"

-x — извлечь только аудио

--audio-format mp3 — указать формат аудио при извлечении

--audio-quality 192K — указать качество аудио

--embed-thumbnail — встроить миниатюру в аудиофайл

--add-metadata — добавить метаданные в файл

--write-subs — скачать субтитры

--write-auto-subs — скачать автоматически сгенерированные субтитры

--sub-lang LANG — указать язык субтитров, например, --sub-lang ru

--playlist-items 1-5 — скачать определённые элементы плейлиста

--download-archive FILE — вести журнал скачанных файлов, чтобы избежать повторных загрузок

--no-playlist — скачать только одно видео, даже если ссылка указывает на плейлист
